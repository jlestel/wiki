# Ollama

Outil pour exécuter des LLMs en local (macOS, Linux, Windows). API REST simple, large bibliothèque de modèles.

## Intérêt

- Expérimenter avec l'IA sans dépendre du cloud
- Brique technique pour homelab et automatisations
