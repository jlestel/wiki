# Fondamentaux de la computer science

Selon [[brivael]], la CS est un art qui repose sur 3 piliers hiérarchiques — [[2026-04-14-apprendre-a-coder]] :

## Les 3 piliers

### 1. Data structures (vocabulaire)

Comment modéliser la data : array, hashmap, arbre, graphe. Le vocabulaire de base pour décrire le monde en code.

### 2. Algorithmes (grammaire)

Comment structurer les opérations : trier, chercher, parcourir, optimiser. La grammaire qui transforme le vocabulaire en phrases qui ont du sens.

### 3. Systèmes (orchestration)

Comment penser à l'échelle : on n'écrit plus du code, on orchestre des systèmes qui wrappent des technologies, elles-mêmes conteneurs de data structures et d'algorithmes.
