# Drew Breunig

Analyste tech, auteur du blog dbreunig.com. Connu pour ses analyses des implications économiques de l'IA.

## Positions

- La cybersécurité est devenue du proof-of-work : on ne gagne pas en étant malin, on gagne en dépensant plus de tokens que l'attaquant
- L'open source reste critique pour la sécurité malgré les arguments pro-réimplémentation via LLM
- Le code est cheap, sauf s'il doit être sécurisé
