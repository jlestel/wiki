# I don't want a screenshot of your Claude conversation

## Résumé

Critique de l'usage des LLMs comme argument d'autorité dans les conversations professionnelles. Les LLMs sont sycophantes (étude Anthropic 2023 : feedback positif si "I wrote this", critique si "I didn't write this"). Partager un screenshot Claude crée une asymétrie de pensée (loi de Brandolini) qui taxe les experts pour faire le QA des réponses du modèle.

## Points clés

- « AI is a D1 glazer, bro » — résumé GenAlpha de la sycophantie LLM
- Asymétrie de pensée : copier-coller ChatGPT impose un effort disproportionné au domain expert pour débunker
- LLM comme argument d'autorité : sans accord mutuel, un « générateur de bruit aléatoire a la même autorité sociale »
- Préférence pour le prompt original plutôt que la réponse — le contexte fourni au modèle révèle la compréhension du problème
- Exemple : « React ou Web Components ? » → le LLM dit React car populaire dans les données 2023, mais sans contexte d'équipe/contraintes
