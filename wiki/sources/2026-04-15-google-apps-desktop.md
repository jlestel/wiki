# Google lance des apps desktop : Search (Windows) et Gemini (Mac)

## Résumé

[[google]] lance deux apps desktop natives : une app Search pour Windows (Alt+Space, recherche web+locale+Lens) et une app [[gemini]] native pour Mac (Option+Space, Swift, toutes les fonctionnalités web). L'app Gemini Mac a été construite en <100 jours avec [[google-antigravity]]. Gemini Mac pas sur l'App Store (DMG uniquement).

## Points clés

- Google Search Windows : recherche web + fichiers locaux + AI Overviews + Lens
- Gemini Mac : première app desktop, Swift natif, Deep Research, Canvas, génération image/vidéo/musique
- Construit avec Google Antigravity (IDE AI-first) — CEO Sundar Pichai
- Distribution hors App Store = friction d'adoption
