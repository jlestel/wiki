# Coolify — alternative open source à Vercel/Netlify/Heroku

## Résumé

[[coolify]] : plateforme de déploiement self-hosted, 53 000+ stars GitHub, Apache-2.0. S'installe sur n'importe quel serveur SSH ($5 VPS, Raspberry Pi). Déploie sites statiques, apps full-stack, BDD, APIs, 280+ services one-click. Fonctionnalités payantes chez les concurrents incluses gratuitement.

## Comparaison coûts

| Plateforme | Coût |
|-----------|------|
| Vercel | $20/seat/mois + surprise bandwidth |
| Netlify | $20/mois |
| Heroku | free tier supprimé |
| **Coolify + Hetzner VPS** | **$5/mois total** (équipe de 5) |

## Fonctionnalités incluses

- SSL auto-renouvelé, backups S3, PR preview deployments
- Terminal serveur real-time, push-to-deploy (GitHub/GitLab/Bitbucket)
- Monitoring avec alertes Discord/Telegram/email
- Zéro vendor lock-in : configs sur le serveur, tout tourne sans Coolify
