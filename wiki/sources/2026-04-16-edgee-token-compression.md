# Edgee — Token Compression Gateway

## Résumé

[[edgee]] est un proxy edge qui compresse les prompts avant qu'ils atteignent les providers LLM. Réduction annoncée jusqu'à 50% des coûts. API compatible OpenAI, 200+ modèles, routing intelligent. S'installe en 1 min, fonctionne comme proxy transparent pour Claude Code, Codex, Cursor.

## Points clés

- Compression de tokens sans perte d'intention
- Sessions de codage 26.5% plus longues
- Edge tools, observabilité, modèles privés (open source serverless)
- BYOK (Bring Your Own Keys)
